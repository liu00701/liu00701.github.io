HTML tags

What does <br> tag mean?
This tag inserts a single line break in a text string.
What does <strong></strong> tag mean?
The tag is used to define important text. The content within it is displayed in bold.
What does <mark></mark> tag mean?
The tag defines text to be highlighted.
What does <q></q> tag mean?
This tag defines a small quotation.
What does <del></del> tag mean?
The tag defines text deleted from a document.
